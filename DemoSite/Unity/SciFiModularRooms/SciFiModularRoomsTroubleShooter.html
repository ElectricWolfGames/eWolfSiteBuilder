Removing White Lines



If you see bits of white lines in the scene



Then we just need to set the camera colour to black.


Light Flares not facing player



When the light flare don't rotate to face the player.



With in the code just update FacePlayer, to be the name of the player object you need the flares to face.


Trying to read Input



If you get the messages

You are trying to read Input using the UnityEngine.Input class, but you have switched active Input handling to Input System package in Player Settings.

Then go to Project Setting -> Player

Update Active Input Handing to use Both

Magenta Models




If you import Sci-Fi Modular Rooms in to a project and find the material are magenta.



We will need to update our material to match the project render pipeline.

Sci-Fi Modular Rooms is using Standard Shader for the materials,

If your project is using Universal Render Pipeline Then we can update our materials

Open the material Rooms001 (eWolf\SciFi_Rooms_Pack1\Art\Materials)

Select Universal Render Pipeline -> Backed Lit shader
Then just update the BaseMap with the Room001_albedo Texture

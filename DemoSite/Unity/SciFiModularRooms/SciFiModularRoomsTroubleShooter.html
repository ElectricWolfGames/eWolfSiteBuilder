Light Flares not facing player



When the light flare don't rotate to face the player.



With in the code just update FacePlayer, to be the name of the player object you need the flares to face.


Trying to read Input



If you get the messagse

You are trying to read Input using the UnityEngine.Input class, but you have switched active Input handling to Input System package in Player Settings.

Then goto Project Setting -> Player

Update Active Input Handing to use Both

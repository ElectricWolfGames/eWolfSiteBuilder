Overview


Most prefabs have the option to randomize the materials used on the objects.


Each material has different colour on a few parts.
You are also free to add more if you wish.

Randomize Colour: Randomize the materials from the list
Set first Color: Use the first material from the list
Set to use bake lights: Only used when/if you add new objects to a prefab and need to set the lighting options.

To randomize all the materials in the scenes, you can use the prefab eWolf/Common/Controller_pf
It has the options

That are applied to all objects.
Randomize All: Randomize all objects in the scene*
Randomize Visual: Randomize the materials from the list
Fix all: Fix all items*
Set the default state: re-set all items*
Set Bake Lighting: Update all objects with to use bake lighting
Generate Lighting: Calls the unity Generate Lighting for the scene.
* Not needed in this package.

Scripts Used


eWolf.SciFiObjects_Pack2.ScifiObject2Base: Base file for most objects, lets you Randomize Visual
eWolf.Common.FacePlayer: Used to rotate an object to face the player.
eWolf.Common.PlaceInRoom: Uses a ServiceLocator, so you can add anything you want it to do here, by just using an IMapController
eWolf.SciFiObjects_Pack2.InteractiveObjects: Just set Operate when you want the object to do it's action. It also saves it's state.
eWolf.SciFiObjects_Pack2.FanBlades: Spins the fan blades, with speed options
eWolf.SciFiObjects_Pack2.Cables: Allows you to add cables
